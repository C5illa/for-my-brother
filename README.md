# for-my-brother
Will it stay that way forever?!

Az elmúlt jó néhány hónapban számos olyan gesztus érkezett felőletek, amiből arra következtettem, hogy nem kívántok kapcsolatba kerülni velem / velünk. Sajnálattal, de tudomásul vettem ezt, és néha nagyon nagy nehézségek árán, de tartottam magam ahhoz, hogy ne zavarjalak benneteket semmilyen formában. Gondoltam, biztos időre van szükségetek, mert a nem mindennapi helyzet, ami elé állított benneteket az élet...
